NES Retro Multiplayer

Both sides run jsnes locally and sync only input over a PeerJS data channel. You can load a ROM from your device or pick from the built-in library.

Host (Player 1)

Run the emulator, load a ROM locally, and sync input in lockstep.

Open Host

Guest (Player 2)

Run the emulator locally and send Player 2 input (keyboard/touch/gamepad).

Open Guest

Phone Controller

Use a phone as a gamepad when the host runs the game on a PC.

Open Controller
